Q.(1) What is JQuery?
=> jQuery is a lightweight, "write less, do more", JavaScript library. The purpose of jQuery is to make it much easier to use JavaScript on your website.


Q.(2) How to Apply CSS Using JQuery, How to Add Class and Remove Class in JQuery,JQuery Animation?
=> 1. addClass() - Adds one or more classes to the selected elements. 
   2. removeClass() - Removes one or more classes from the selected elements.
   3. toggleClass() - Toggles between adding/removing classes from the selected elements.
   4. css() - Sets or returns the style attribute.
